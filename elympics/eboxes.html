E-BOXES

Where it all happens.

Berlin already has the right infrastructure to host the E-lympics. Developed by researchers at the Berlin Institute of Technology, the BrainBox is an interactive environment that allows the creation of virtual reality scenarios. It consists of a cubic structure with a default size of 41m2, which has the possibility to be expanded.

Treadmills for virtual gaming can be placed in the internal corners of the Box, allowing participants to engage in virtual versions of the Olympic Games, such as running, boxing and fencing. Viewers would not only be able to observe the players but also to follow the virtual match in giant screens. The structure of the Box allows Images can be projected onto the whole extension of all four internal walls. A powerful sound system contributes for a true group experience. Equipped with gadgets, such as virtual glasses and high tech shirts, individuals at the Box could put themselves in the place of the athletes.

Designed to be “plug and play”, the Brain Box can be set up anywhere in any city. The only requirements are fast internet and electricity.
